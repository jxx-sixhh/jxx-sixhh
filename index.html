Map

Map 对象保存键值对，并且能够记住键的原始插入顺序。任何值(对象或者原始值) 都可以作为一个键或一个值。

Map()

Map.prototype.set()

set() 方法为 Map 对象添加或更新一个指定了键（key）和值（value）的（新）键值对。

myMap.set(key,value);

返回一个Map 对象。

Map.prototype.get()

get() 方法返回某个 Map 对象中的一个指定元素

myMap.get(key);

返回一个 Map 对象中与指定键相关联的值，如果找不到这个键则返回 undefined。

Map.prototype.has()

方法has() 返回一个bool值，用来表明map 中是否存在指定元素.

myMap.has(key);

如果指定元素存在于Map中，则返回true。其他情况返回false

什么是Redux？

Redux是一个使用叫做’action’的事件来管理和更新应用状态的模式和工具库。它以集中式Store的方式对整个应用中使用的状态进行集中管理。

为什么要使用它

Redux可以帮我们管理‘全局’状态 也就是应用程序共同需要的状态
Redux提供的模式和工具让我们更容易理解应用程序中的状态何时、何地、为什么以及如何更新，以及当这些状态更新时，我们的应用程序逻辑将如何表现。

应该在什么时候使用它？

  • 在应用的大量地方，都存在大量的状态
  • 应用状态会随着时间的推移而频繁更新
  • 更新该状态的逻辑可能很复杂
  • 中型和大型代码量的应用，很多人协同开发

Action

action是把数据从应用传到store的有效载荷，action内会包含一个type字段，来表示将要执行的动作。我们应该尽量减少在action中传递的数据。

action创建函数

Action 创建函数 就是生成 action 的方法。

Reducer

Reducers 指定了应用状态的变化如何响应 actions 并发送到 store 的，记住 actions 只是描述了有事情发生了这一事实，并没有描述应用如何更新 state。

reducer 就是一个纯函数，接收旧的 state 和 action，返回新的 state。

(previousState,action) => newState

保持reducer纯净非常重要,不要在reducr中做以下这些事

  • 修改传入参数
  • 执行有副作用的操作，如api请求和路由跳转
  • 调用非纯函数，如Date.now()或者Math.random()

谨记 reducer 一定要保持纯净。只要传入参数相同，返回计算得到的下一个 state 就一定相同。没有特殊情况、没有副作用，没有 API 请求、没有变量修改，单纯执行计算。

注意点

store必须是唯一的，只能有一个store空间

只有store能改变自己的内容，reducer不能改变

Reudcer只是返回了更改的数据，但是并没有更改store中的数据，store拿到了Reducer的数据，自己对自己进行了更新。

reducer必须是纯函数

什么是纯函数？

如果函数的调用参数相同，则永远返回相同的结果。它不依赖于程序执行期间函数外部任何状态或数据的变化，必须只依赖于其输入参数。

比方说在我们写的这段reducer代码中，reducer的返回结果，完全是由传入的参数state和action决定的，这就是一个纯函数

JSX

  1. react介绍JSX

    JSX是JS的一个语法扩展，是React.createElement函数的语法糖

  2. react里面JSX直接绑定事件有啥缺点

    ⭐第一种：onClick = {this.yourEvent} 需要在构造函数中绑定this

     第二种：`onClick = {this.yourEvent.bind(this)}` //每次render,都会重新生成一个函数
 第三种：`onClick = {() => this.yourEvent()}` //每次render，都会执行一遍方法

  3. JSX的本质;

    实际上JSX是React.createElement(component,props,...children)函数的语法糖

  4. 为什么在JSX里循环生成DOM需要添加key

    在进行diff算法时，如果没有key的话，在列表数据渲染过程中，如果我后来又在前面又添加了一条数据，那么所有他后面的数据都要被修改。如果有key的话，react将根据key属性匹配原有树上的子元素以及最新树上的子元素，这样的话只需要把最新的那个元素插入到最前面位置即可。

  5. 浏览器怎么识别JSX的语法

    浏览器是无法识别JSX的，所以我们需要依靠babel或者ts来将JSX转换为JS。在react17之前，都是把他转换成react.createElement()函数，在react17后，它是自动从 React 的 package 中引入新的入口函数并调用。

  6. 浏览器不能识别JSX文件，需要怎么办？

    配置@bebel/preset-react,用于处理JSX类型的文件，将其转换为JS文件

  7. JSX怎么转成原生JS。JSX怎么用React.createElement创建?

    JSX与JS在线转换器

parseInt(string,radix)

该函数解析一个字符串并返回指定基数的十进制整数

参数

string 必需

返回值

截断解析

当radix是undefined、0、或者压根没有的时候，js会假定以下情况。
如果string以0x(x无论大小写)开头，那么radix会假定为16。
如果string以0开头，那么radix会假定为8/10。。ES5澄清了会假定为10，但是！并不是所有浏览器都支持。因此，在使用parseInt时，一定要指定一个radix

1. 双指针

ex1:合并两个有序数组


看到这道题你可能跟我一样首先想到的是先合并后排序，不过面试的时候要是敢这么写，面试官可能当场就说“好，今天的面试就到这里了”。后来或许我们可以想到再新建一个数组，利用双指针比较nums1和nums2头部的数字，取出比较小的存在新数组里。不过，这种方法还是可以继续优化。因为我们可以看到nums作为结果数组，其本身的长度是足够长的，所以我们可以将指针从右往左倒序遍历，取出比较大的放在nums1的后部。这样空间复杂度便可变为O(1)，因为我们没有占用额外空间
