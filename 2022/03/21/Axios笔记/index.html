axios是什么？

Axios是一个基于promise的HTTP库，可以利用它发送get、post请求。说到这里你可能会想，不是有ajax了吗？我们为什么还要学习axios呢？
那我们再回顾一下ajax是做什么的。Ajax是为了实现网页异步更新而产生的，也就是说，使用了ajax，我们可以在不重新加载整个页面的情况下，对网页的某个部分进行更新。那么实现Ajax都有哪些方法呢？

  1. 使用jQuery封装的ajax($.ajax()).值得一提的是，因为react和vue的出现，前端工程师不必直接操作dom，也就是说，jquery如今已经没那么吃香了，那我们为了ajax而引入jQuery.js就显得没什么必要了。
  2. 原生的xhr 原生的xhr有多繁琐我们也再回顾一下
  3. Axios，其本质其实还是对xhr的封装。不过因为它基于Promise，所以是符合最新的es规范的。它具备以下特点
  • 基于XHR+Promise的异步Ajax请求库。因此它支持Promise API
  • 在浏览器端和node端都可以使用
  • 支持请求 / 响应拦截器
  • 支持请求取消
  • 支持请求 / 响应数据转换
  • 自动转换json数据
  • 可批量发送多个请求

拦截器

拦截器是指当发送请求或者得到响应被then或catch处理之前对它们进行拦截，拦截后可对数据做一些处理，比如给请求数据添加头部信息，或对响应数据进行序列化，然后再传给浏览器，这些都可以在拦截器中进行
